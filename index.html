Projets réalisés

Application Android Météo

Projet L3 IUP Brest • Novembre 2014 - Janvier 2015

Projet en équipe de deux étudiants consistant à réaliser une application android où l'utilisateur puisse consulter des données météorologiques. Le but étant de développer une application intuitive grâce à une phase d'analyse. L'application a été développée sur Eclipse, en Java avec le SDK Android. Les données à afficher ont été récupérées grâce à une API (OpenWeatherMap).
